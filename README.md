# Gestión de Clientes

Componente desarrollado en React / Next.js para la gestión básica de clientes.

## Funcionalidades
- Alta de clientes
- Eliminación
- Búsqueda por nombre o email
- Manejo de estado con hooks

## Notas
Datos manejados en memoria, sin backend.
Este repositorio contiene únicamente el componente principal del frontend
(formulario y listado de clientes) como prueba de concepto.
No incluye la estructura completa de un proyecto Next.js.
